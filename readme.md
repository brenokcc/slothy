# Slothy

## Dependencies

### Python
- six
- django
- django_rest_framework

### Nodejs
- sync-request

## Installation

### Python

> pip install slothy


## Creating a Project

### Back-end

>python -m slothy back-end < name >
>
> cd < name >
>
>python manage.py runserver


### Front-end

>python -m slothy front-end < name >
>
>cd < name >
>
>python manage.py runserver

## Clients

### Nodejs

>npm install slothy-nodejs

> node
> var slothy = require('slothy-nodejs/slothy')
